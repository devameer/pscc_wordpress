# دليل الترجمات | Translation Guide

## نصوص صفحة البحث | Search Page Strings

تم تسجيل جميع النصوص التالية في نظام Polylang. يمكنك ترجمتها من لوحة تحكم WordPress.

### كيفية الترجمة | How to Translate

1. اذهب إلى: **WordPress Admin → Languages → String translations**
2. ابحث عن النصوص أدناه
3. أضف الترجمة العربية لكل نص
4. احفظ التغييرات

## ملاحظات مهمة | Important Notes

### رموز التنسيق | Formatting Codes

بعض النصوص تحتوي على رموز خاصة:

- **%s** = يتم استبداله بنص (مثل كلمة البحث)
- **%d** = يتم استبداله برقم (مثل عدد النتائج)
- **%1$s**, **%2$s** = نصوص متعددة بترتيب معين

**مهم جداً:** احتفظ بهذه الرموز في الترجمة العربية!

### مثال:

```
English: Search Results for: %s
Arabic: نتائج البحث عن: %s
```

4. ✅ التحقق من جميع الصفحات:
   - الصفحة الرئيسية
   - صفحة البحث

   - صفحة المعرض
