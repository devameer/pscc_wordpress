# دليل إعداد الحقول متعددة اللغات

تم إنشاء جميع الحقول المطلوبة تلقائياً ✅

## الحقول التي تم إنشاؤها

تم إنشاء الحقول التالية تلقائياً من خلال ملف JSON:

### 1. تسمية البحث
- **الحقل الإنجليزي**: `topbar_search_label` (حقل نصي)
- **الحقل العربي**: `topbar_search_label_ar` (حقل نصي)

### 2. رابط التبرع / زر الإجراء
- **الحقل الإنجليزي**: `donate_link` (حقل رابط - يشمل URL والنص والهدف)
- **الحقل العربي**: `donate_link_ar` (حقل رابط - يشمل URL والنص والهدف)

### 3. رابط الأسئلة الشائعة
- **الحقل الإنجليزي**: `faq_link` (حقل رابط - يشمل URL والنص والهدف)
- **الحقل العربي**: `faq_link_ar` (حقل رابط - يشمل URL والنص والهدف)

## كيفية الوصول إلى الحقول

**لا حاجة لإنشاء الحقول يدوياً!** الحقول محفوظة في:
```
resources/acf-json/group_theme_options_multilingual.json
```

سيقوم ACF بمزامنة هذه الحقول تلقائياً عندما:
1. تذهب إلى **WordPress Admin > Custom Fields**
2. سترى إشعاراً حول مجموعات الحقول المطلوب مزامنتها
3. انقر على "Sync available" لاستيراد الحقول

أو سيتم تحميل الحقول تلقائياً عند تحميل الصفحة التالية.

## كيفية تعبئة القيم

1. اذهب إلى **لوحة تحكم WordPress > Theme Settings**
2. ستجد تبويبين:
   - **English Settings**: للقيم الإنجليزية
   - **Arabic Settings / الإعدادات العربية**: للقيم العربية
3. املأ كل حقل:

### مثال - القيم الإنجليزية:
- **Search Label**: "Search"
- **Donate Link**:
  - عنوان URL: https://example.com/donate
  - نص الرابط: "Donate"
  - الهدف: Default
- **FAQ Link**:
  - عنوان URL: https://example.com/faq
  - نص الرابط: "FAQs"
  - الهدف: Default

### مثال - القيم العربية:
- **تسمية البحث**: "بحث"
- **رابط التبرع**:
  - عنوان URL: https://example.com/ar/donate
  - نص الرابط: "تبرع"
  - الهدف: Default
- **رابط الأسئلة الشائعة**:
  - عنوان URL: https://example.com/ar/faq
  - نص الرابط: "الأسئلة الشائعة"
  - الهدف: Default

## كيف يعمل النظام

الموقع يكتشف اللغة الحالية تلقائياً ويستخدم الحقول المناسبة:

- **الإنجليزية (en)**: يستخدم الحقول بدون لاحقة
- **العربية (ar)**: يستخدم الحقول مع لاحقة `_ar`
- **نظام الاحتياط**: إذا كان الحقل الخاص باللغة فارغاً، يستخدم الحقل الإنجليزي

## إضافة لغات إضافية

لإضافة دعم للغات أخرى (مثل الفرنسية - fr):

1. أضف حقول ACF جديدة مع لاحقة اللغة: `donate_link_fr`, `faq_link_fr`, `topbar_search_label_fr`
2. الكود في `header.php` سيستخدم هذه الحقول تلقائياً عند عرض الموقع بالفرنسية

## الملفات المعدلة

### 1. Header Template
**الملف**: `resources/views/layout/header.php` (الأسطر 40-63)

يكتشف اللغة الحالية ويحمل الحقول المناسبة.

### 2. ACF Field Group
**الملف**: `resources/acf-json/group_theme_options_multilingual.json`

يحتوي على تعريف جميع الحقول.

### 3. Language Functions
**الملف**: `app/languages.php`

يحتوي على جميع دوال الترجمة وتسجيل نصوص Polylang.

## الدعم الفني

إذا واجهت أي مشاكل:
1. تأكد من تفعيل إضافة ACF
2. تأكد من تفعيل إضافة Polylang
3. اذهب إلى Custom Fields وتحقق من وجود Field Group اسمه "Theme Options - Multilingual"
4. إذا لم يظهر، انقر على "Sync" لمزامنة الحقول من ملف JSON
